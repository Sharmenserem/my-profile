1. Requirements

Make sure your server meets the following requirements:

  • PHP 8.1 or higher
  • file_get_contents function enabled
  • cURL support enabled

2. Ads Setup

To enable or disable ads, edit the config.php file:

Enable/Disable Ads

define("ADS", true); // Enable ads
define("ADS", false); // Disable ads

Ad Placement Configuration

Edit these parts inside config.php to set your ads:

$config = array(
    "ad728" => '<img src="https://dummyimage.com/728x90/E1306C/fff" class="img-fluid">',
    "ad468" => '<img src="https://dummyimage.com/468x60/833AB4/fff" class="img-fluid">',
    "ad300" => '<img src="https://dummyimage.com/300x250/833AB4/fff" class="img-fluid">'
);

Replace the dummy image URLs with your real ad codes (like AdSense banners).

3. RapidAPI Setup

This script uses RapidAPI to fetch Instagram media data.

Use This RapidAPI Service

https://shorturl.at/KyjM9

Note: Use this RapidAPI because the script is specially configured based on its API response structure. Using a different RapidAPI may break functionality.

Update RapidAPI URL

Edit instasave.core.php and update this line:

// RapidAPI endpoint URL
$rapidapi_url = "your-rapidapi-Url" . urlencode($this->url);

Update RapidAPI Headers

Also, update your API Host and Key:

// Set headers for RapidAPI
$headers = array(
    'x-rapidapi-host: your-rapidapi-host',
    'x-rapidapi-key: your-rapidapi-key'
);

Make sure your API key is active and you have selected a suitable RapidAPI plan.

Follow these steps to install and run the script:

  1. Upload Files: Upload all extracted files into your hosting public_html or htdocs folder.
  2. Set Permissions: Ensure PHP files have at least read permission (644).
  3. Configure Ads: Edit config.php to manage ad settings if needed.
  4. Setup RapidAPI: Edit instasave.core.php and set your RapidAPI Host and Key.
  5. Access Script: Open your domain in the browser to test the script (example: https://yourdomain.com/).
